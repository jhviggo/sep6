## How to test and run
Currently the only tests are linting by [eslint](https://www.npmjs.com/package/eslint) with very simple rules.

```
# Test
npm run test

# Run dev server
npm run dev

# Run server
npm start
```

## Set up environment
Create a `.env` file and place the secrets from firebase into their respective variables
```
API_KEY=""
APP_ID=""
```
